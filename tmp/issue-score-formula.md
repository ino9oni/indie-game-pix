# Issue: スコア計算式を残時間×1へ変更

## 背景
- 現在の加点は「残秒 × 100 + クリアボーナス」で、想定よりもスコアが膨らみやすい。
- GAMEDESIGNの更新に合わせ、係数を1に変更してゲームバランスを整える必要がある。

## 要件
- ピクロスクリア時のスコアを「残時間（秒数）×1 + クリアボーナス」で算出するよう修正する。
- GAMEDESIGNの該当節を新しい式に差し替え、仕様の一貫性を保つ。
- スコア加算アニメーションと効果音は従来どおり使用し、演出時間が極端に短くならないよう調整する。
- 小数秒は切り捨てて整数で計算する。

## 完了条件
- クリア時に加算される数値が新式に基づくことを UI 上で確認できる。
- 残秒0の場合はクリアボーナスのみ加算される。
- GAMEDESIGN と実装の内容が一致している。

# Issue: ピクロス対戦モードの実装

## 背景
- TASKLIST の高優先タスクとして、プレイヤーと敵が同時にピクロスを解く対戦モードとスペル要素が求められている。

## やること
- ピクロス画面を左右2面レイアウトに変更し、プレイヤー/敵を同時表示する。
- 敵AI（難易度別の解答速度・誤答率）を実装し、3問連続の勝負で競わせる。
- HEROおよび各敵のスペル（発動条件・効果）を実装し、演出/ログを追加する。
- 勝敗判定（3問先取）、進捗バー、スペルログ等のUI要素を追加する。
- GAMEDESIGN と実装内容を同期する。

## 完了条件
- プレイヤーと敵が同時進行でピクロスを解き、先に3問クリアした側で勝敗が決まる。
- スペルの発動条件・効果が仕様通りに動作し、演出／ログが表示される。
- 仕様ドキュメントに対戦モードのレイアウト、AIパラメータ、スペル仕様が追記されている。

# Issue: ピクロス対戦に一時停止/再開を追加する

## 背景
- 対戦中に離席したい場合でもタイマーと敵AIが進み続けてしまい不便だった。

## 要件
- トップアクション領域に `Pause` ボタンを追加し、対戦中のみ押下できるようにする。
- `Pause` 押下時はタイマーと敵AIの進行を停止し、盤面操作を無効化する。
- 一時停止中は画面中央に `Paused` オーバーレイと `Resume` ボタンを表示する。
- `Resume` 押下でタイマーと敵AIが再開し、盤面操作も元に戻る。
- GAMEDESIGN と TASKLIST を最新仕様へ更新する。

## 完了条件
- ピクロス対戦で `Pause` → `Resume` による一時停止/再開が機能する。
- 一時停止中は盤面操作ができず、敵AIやタイマーも停止する。
- ドキュメントが実装内容と一致する。

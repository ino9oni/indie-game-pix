# Issue: ピクロス盤面の塗り分けボタンを撤去する

## 背景
- 操作用の `Fill` / `Cross` / `Maybe` トグルがヘッダー領域を占有し、画面が窮屈に見える。
- クリック操作だけで同じ入力ができるため、ボタンを残す必要がなくなった。

## 要件
- 対戦画面から塗り分けトグルボタンを削除する。
- 左クリック/右クリック/Shift+左クリックで従来通りの塗り分けが行えるよう維持する。
- ドキュメントに操作手段を追記し、仕様を最新化する。

## 完了条件
- 画面上にトグルボタンが表示されない。
- マウス操作のみで Fill / Cross / Maybe の切り替えが可能。
- GAMEDESIGN とタスク仕様が更新されている。

# タスク仕様: トータルスコア表示をヘッダー右上へ固定

## 背景
- 現状のトータルスコア HUD はヘッダー左側のブランドロゴ下に表示され、他の操作ボタンと高さ・配置が揃っていない。
- TASKLIST の指示では、スコアを常に画面右上の操作ボタン群と同じサイズ・位置で表示する必要がある。

## 目的
- スコアをトップアクション領域（右上）に統合し、他のボタンと一体感のあるレイアウトに揃える。

## 要件
1. 全モードでヘッダー右上（`top-actions`）へスコア表示を配置し、他のボタンと同じ高さ／余白（ボタンと同等のタッチ領域）にする。
2. スコア表示は常に表示し、加算アニメーション中も同位置で動作する。
3. 既存の ScoreDisplay コンポーネントを再利用しても良いが、外観はコンパクトなチップ型に調整する。
4. GAMEDESIGN のグローバル UI 仕様に、右上固定表示のルールを追記する。

## テスト観点
- どの画面でもスコアが右上の操作ボタン列と揃ったサイズで表示される。
- フォントサイズ切替・Pause・BGM ボタンなどの操作とレイアウトが崩れない。
- スコア加算アニメーションが右上表示のまま問題なく動作する。

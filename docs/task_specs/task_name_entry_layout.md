# タスク仕様: 名前入力モードの画面レイアウト変更

## 背景
- TASKLIST にて名前入力モードの操作ボタン配置変更が指示されている。
- 現状は全ての操作ボタンが縦一列に並び、決定ボタンの位置がプロローグの "Next" ボタン位置と揃っていない。

## 目的
- 操作ボタン配置を再調整し、決定ボタンをプロローグの Next ボタンと同じ画面右下位置に配置する。
- 左右の移動ボタンを画面下段にまとめ、操作性と視認性を高める。

## 要件
1. 名前入力モード（独立画面およびゲーム開始フロー内）で表示される `NameEntry` コンポーネントのボタンレイアウトを変更する。
2. 決定ボタンはカード右下に固定し、プロローグの Next ボタン位置と揃える。
3. 左右の移動ボタン（`←`, `→`）は決定ボタンと同じ下段の横並びに配置する。
4. 一文字削除や戻るボタンなどその他操作は下段とは分け、上段に縦並びで配置する。
5. レイアウト変更後もキーボードフォーカス順と既存の処理ロジック（アクション関数）は維持する。
6. GAMEDESIGN を更新して新しいレイアウト指示を明文化する。

## テスト観点
- 名前入力モードでボタンが上下に分かれて配置され、決定ボタンが右下に位置する。
- プロローグのゲーム開始フロー内でも同レイアウトになる。
- 他モードの UI や挙動に影響がない。

# Task Spec: ピクロス対戦盤面に会話用ポートレートを表示する

## 背景
- 会話モードではキャラクターのポートレートが表示されるが、ピクロス対戦画面には表示されていない。
- 対戦の臨場感を高めるため、各陣営の盤面付近に会話用ポートレートを常時表示したい。

## 要件
- プレイヤー側の盤面に主人公のポートレートを表示する。
  - 画像は会話モードと同じ `assets/img/character/hero/hero_normal.png` を使用する。
  - サイズは高さ100px固定で、横幅は画像の縦横比を保って自動調整する。
  - 表示位置はプレイヤー盤面の左上（ヒント領域に重ならない位置）とし、盤面と少し重なる形でカード風にレイアウトする。
- 敵側の盤面に敵キャラクターのポートレートを表示する。
  - 表示する画像はそのノードの会話で使用する通常立ち絵（`CHARACTERS[difficultyId].images.normal`）。
  - サイズ・配置はプレイヤー側と同様（高さ100px、縦横比維持、盤面左上）。
- ポートレートには淡い境界線とドロップシャドウを付与し、背景と区別が付くようにする。
- ピクロス対戦以外の画面では表示しない。

## 完了条件
- プレイヤー・敵双方の盤面に指定のポートレートが表示される。
- 盤面サイズに応じてヒントやセルと干渉せず、レスポンシブに位置・サイズが確保される。
- GAMEDESIGN と TASKLIST に要件が反映される。

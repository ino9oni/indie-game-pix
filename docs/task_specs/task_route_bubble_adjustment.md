# タスク仕様: マップ吹き出しのサイズ調整

## 背景
- 新しい地点名称を吹き出しで表示したが、文字が切れるケースがある。
- TASKLIST では吹き出しを横長にし、2-3px のパディングを確保することが求められている。

## 目的
- 吹き出し内の文字が収まりやすいよう横幅と余白を調整し、視認性を向上させる。

## 要件
1. 各地点名称に対して十分な横幅を確保する（文字数×16px 以上を目安とする）。
2. ラベル内のテキストと境界の間に 2〜3px のパディングを設ける。
3. 吹き出しの下向き三角形も新しい横幅に合わせてバランス良く配置する。
4. スタイル定義（CSS）を更新し、ラベル色・輪郭を統一する。
5. GAMEDESIGN に吹き出しレイアウトの概要が記載されていることを確認する。

## テスト観点
- すべての地点名称が切れずに表示される。
- 吹き出しと三角形の位置関係が自然に見える。
- フォントスケール変更時にも余白が保たれている。

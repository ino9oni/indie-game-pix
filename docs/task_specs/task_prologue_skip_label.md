# タスク仕様: プロローグスキップボタンの英語表記統一

## 背景
- プロローグ画面のスキップボタンは現在「スキップ」と日本語表記になっている。
- TASKLIST 改善項目で、ボタン名称を「Skip」に改め、オープニング画面の「New Game」と同じスタイルでそろえるよう求められている。

## 目的
- 画面内の主要 CTA を英語で統一し、全体の UI テイストと合わせる。

## 要件
1. プロローグ画面のスキップボタン文言を「スキップ」から「Skip」に変更する。
2. ボタンの見た目（コンポーネント／クラス割り当て）はオープニング画面の「New Game」ボタンと同一のスタイルを適用する。
3. GAMEDESIGN にボタン名称が「Skip」である旨を追記する。

## テスト観点
- プロローグ表示中にボタンが「Skip」と表示されている。
- オープニング画面の「New Game」ボタンと同じスタイル（色・角丸・ホバー挙動）が適用されている。
- クリック時に従来どおりプロローグがスキップされ、遷移挙動が変化していない。

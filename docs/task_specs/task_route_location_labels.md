# タスク仕様: マップモード地点名称と吹き出し表示

## 背景
- 既存のマップモードではノードの下に難易度名（Practice/Easy など）が表示されており、世界観に即した地点名称が示されていない。
- TASKLIST の高優先度項目で、地点名称を吹き出し表示に切り替えることが求められている。

## 目的
- 各ノードに独自のロケーション名を表示し、プレイヤーの没入感を高める。
- 表示は吹き出しスタイルとし、視認性を向上させる。

## 要件
1. `RouteMap` のラベル描画を吹き出し型（角丸矩形＋下向き三角形）に変更する。
2. 表示文言を以下の名称へ置き換える。
   - elf-practice: 翠緑の門
   - elf-easy: 木漏れ日の小径
   - elf-middle: 刻印の広間
   - elf-bad-ending: 儚き灯火
   - elf-hard: 精霊の花園
   - elf-ultra: 森羅の社
   - elf-true-ending: 黎明の庭
3. 吹き出しには上下左右1px相当の余白を確保し、テキストは中央揃えで表示する。
4. GAMEDESIGN のルート仕様に地点名称と吹き出しレイアウトを明記する。

## 非機能要件
- フォントスケール変更時にもラベルが崩れないこと。
- 既存のノードクリック判定やアニメーションに影響を与えないこと。

## テスト観点
- 各ノードで新しい名称が表示される。
- 吹き出しの形状が指定通り（角丸＋下向き三角）になっている。
- フォントスケール100/150/200% でもレイアウトが崩れない。

# Task Spec: ピクロスモードの塗り分けボタンを削除する

## 背景
- 現在のピクロス盤面には `Fill` / `Cross` / `Maybe` のトグルボタンが並び、操作スペースを圧迫している。
- マウス操作（左クリック・右クリック・Shift+クリック）で同等の入力が可能なため、ボタンを廃止してシンプルなレイアウトにしたい。

## 要件
- ピクロス対戦画面から `Fill` / `Cross` / `Maybe` のトグルボタン群を削除する。
- 盤面の操作手段は以下のとおり維持する。
  - 左クリック: マスを塗り（Filledに）／再度クリックで解除。
  - 右クリック（またはコンテキストメニュー）: バツ印（Cross）を付与／再度で解除。
  - Shift+左クリック: Maybe マークの切り替え。
- UI からボタンを除外したことで生じる余白は自動的に調整し、盤面やヒントの位置が崩れないようにする。
- ドキュメント（GAMEDESIGN）を操作仕様に合わせて更新する。

## 完了条件
- トグルボタンが表示されず、マウス操作のみで従来通り塗り分けが行える。
- レイアウトが崩れない（盤面、ヒント、ポートレートなどが正常表示される）。
- GAMEDESIGNの操作記述が最新になっている。

# タスク仕様: 名前入力画面後の背景差し替え

## 背景
- GAMESTART モードで名前を確定した後（後半テキストの確認パート）では背景がランダム化されたままで、タスク指示の `assets/img/character/hero/hero_fullbody.png` と一致していない。

## 目的
- 名前入力前後で連続した演出を維持し、後半テキスト表示中も主人公の全身画像を背景に固定する。

## 要件
1. `screen === "gamestart"` かつ名前入力後（フェーズ `after`）でも `assets/img/character/hero_fullbody.png` を背景として表示する。
2. 背景切替は既存の `Background` コンポーネント経由で行い、他モードへの影響を避ける。
3. GAMESTART セクションの仕様（GAMEDESIGN）に、名前入力後も同背景を維持する旨を追記する。

## テスト観点
- 名前入力を完了し後半の導入テキストを読んでいる間も、背面に主人公全身画像が表示される。
- ルート画面へ遷移すると従来のマップ背景へ戻る。
- 他モードの背景挙動に副作用がない。

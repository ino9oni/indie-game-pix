# タスク仕様: hard / ultra ピクロス答案パターンの更新

## 背景
- hard と ultra の問題セットは直線や塗りマスが少なく、ヒント密度が低いため解きごたえに欠ける。
- TASKLIST 改善項目では、より複雑で直線が多い答え画像に差し替えることが求められている。

## 目的
- hard/ultra の問題に多段の直線と左右対称要素を加え、ヒント本数を増やして高難度らしさを出す。

## 要件
1. 20×20 (hard) のパターンを3種類差し替える。
   - 長い水平帯・縦帯を複数含み、ヒントが増える構成とする。
2. 25×25 (ultra) のパターンを3種類差し替える。
   - 枝分かれや円環など複雑なシルエットを持たせ、視覚的にも満足度の高い図形にする。
3. `src/game/puzzles.js` の `HARD20` `ULTRA25` 配列を更新する。
4. GAMEDESIGN に出題モチーフを追記し、仕様と実装が一致するようにする。

## テスト観点
- hard/ultra のピクロスが新しいシルエットで描画される。
- 各行列のヒント数が旧版より増え、長めの連続マスが含まれている。
- ランダム選択処理や既存の難易度配列への参照が壊れていない。

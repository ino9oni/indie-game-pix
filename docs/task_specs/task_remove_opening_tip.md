# タスク仕様: オープニング画面のTip削除

## 背景
- オープニング画面に表示している「Tip: サウンドは右上のトグルで切り替えできます。」という文言は、現在のUIでは自明な情報であり、画面レイアウト上のノイズになっている。
- TASKLIST の改善項目として当該文言の削除が求められている。

## 目的
- オープニング画面をよりシンプルにし、不要なテキストを排除する。

## 要件
1. `Opening` コンポーネントから当該テキスト行を削除し、スタイルの余白や位置が不自然にならないよう調整する。
2. GAMEDESIGN のオープニングに関する仕様からも、Tip 文言に関する記述があれば削除して整合性を保つ。
3. その他の要素（タイトル、ボタン類、テキストアニメーションなど）には影響を与えない。

## 非機能要件
- 文言削除後もコンポーネントのアクセシビリティおよびレスポンシブレイアウトが維持されること。

## テスト観点
- オープニング画面に Tip 文言が表示されないことを目視確認。
- 既存のボタン配置やキーボード操作が変わっていないことを確認。

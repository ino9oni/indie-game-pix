# Task Spec: スペル発動時の攻撃エフェクトとSE

## 背景
- スペル発動時にログとオーバーレイは表示されるが、相手に攻撃する演出がなく迫力に欠ける。
- 指示では発射エフェクトと攻撃的な効果音を追加することが求められている。

## 要件
- HERO のスペル発動時、敵盤面に向かって光の矢のようなエフェクトが飛ぶ演出を表示する。
- ENEMY のスペル発動時、プレイヤー盤面に対して赤系の攻撃エフェクトを表示する。
- エフェクトは 1 秒程度で完結し、盤面操作を妨げない（クリックは可能なまま）。
- スペル発動時に攻撃的な SE を 1 回だけ再生する。
  - 新規 SE ファイル: `assets/se/spell_attack.wav`（仮定）。
- GAMEDESIGN に演出仕様を追記する。

## 完了条件
- スペル発動時にビジュアルエフェクトと攻撃SEが再生される。
- プレイヤー/敵どちらのスペルでも適切な方向へエフェクトが表示される。
- ドキュメント更新済み。

# Task Spec: ピクロス自動遷移（Submit不要）

## 背景
- 既存仕様ではプレイヤーが **Submit** ボタンを押して結果判定を行う必要があり、完全に正解していても操作が1ステップ増える。
- タスク指示では、ピクロス盤面が解答と完全一致した瞬間に自動でクリア扱いにし、同レベル内の次パズルへ遷移することが求められている。

## 要件
- ピクロス盤面の状態が解答と完全一致した場合、**Submit** ボタンを押さなくても直ちにクリア処理を実行する。
- クリア処理では現在の残り時間を用いたスコア加算と勝利カウントの更新を従来通り行う。
- 自動クリア発生時もスコア演出や次パズルへの遷移タイミングは手動Submitと同一にする。
- パズルが複数ある場合は同レベル内で次のパズルを自動的にロードする。最後のパズルを解いた場合はバトル勝利扱いとする。
- 盤面が再度編集された場合は、解答と一致しなくなった時点で自動クリア待機状態を解除する。

## 完了条件
- 盤面を正しく完成させた瞬間に、自動的にクリア演出・スコア加算が行われる。
- パズルを解いている途中で誤ったマスがある場合は自動クリアが発動しない。
- ドキュメント（GAMEDESIGN, TASKLIST）に仕様が反映されている。

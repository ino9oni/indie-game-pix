# Task Spec: ピクロス対戦盤面のサイズ統一とレスポンシブ調整

## 背景
- 現状の対戦画面ではプレイヤー盤面（GameBoard）と敵盤面（EnemyBoard）の表示サイズが僅かに異なり、レイアウトの一貫性が崩れている。
- 画面サイズによっては一方が縦長になったり、余白が不均一になるケースがある。

## 要件
- HERO（プレイヤー）と ENEMY のピクロス盤面を常に同じ幅・高さで表示する。
- ウィンドウ幅に応じて両盤面が等比で縮小・拡大し、縦横比（正方形）が保持されるようにする。
- 盤面のサイズ調整に合わせてヒント・ポートレートも自然に収まるよう余白を再調整する。
- レイアウト変更によりスクロールが発生しないよう、モバイル〜デスクトップまでレスポンシブ対応とする。
- ドキュメント（GAMEDESIGN）にレイアウト仕様を反映する。

## 完了条件
- 各デバイス幅でプレイヤーと敵の盤面サイズが一致する。
- 縦横比が崩れず、ヒント文字・ポートレートが適切な位置に表示される。
- GAMEDESIGN のレイアウト節に仕様が追記されている。

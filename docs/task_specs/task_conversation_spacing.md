# タスク仕様: 会話モードのポートレート配置調整

## 背景
- 現状の主人公／敵ポートレートは中央寄りに配置されており、双方が近すぎる印象になっている。
- 会話ウィンドウと重ならない範囲で左右に寄せ、余白バランスを改善したい。

## 要件
1. 会話モードのポートレート領域を左右に約 4〜6vw 程度の余白を確保しつつ配置する。
2. 画面サイズに応じてレスポンシブに縮小し、小画面では縦スタックへ切り替える。
3. 会話ウィンドウとポートレートが重ならないように間隔（gap）を設ける。
4. GAMEDESIGN の会話モード仕様に、左右余白を確保する旨を追記する。
5. 既存の発光／拡大演出は維持する。

## テスト観点
- PC 幅で左右端に程よく寄り、中央に十分なスペースがある。
- スマートフォン幅では重なりなく縦に並ぶ。
- 会話ウィンドウやスキップボタンとレイアウトが干渉しない。

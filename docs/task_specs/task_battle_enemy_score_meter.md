# タスク仕様: バトルモードのスコアメーター整備

## 背景
- 既存UIではヘッダーにプレイヤースコアを表示しているだけで、敵スコアは可視化されていない。
- TASKLIST により、バトルモードの各陣営パネル内でスコアを表示する配置が求められている。

## 目的
- HERO / ENEMY 双方にスコアメーターを用意し、スペルスロット右側に配置して常時表示する。
- バトル用スコア表示をヘッダーから排除し、モード内で完結させる。

## 要件
1. プレイヤー側パネル: スペル枠右端に `Score` ラベル + 累積スコア数値（カンマ区切り）を縦並びで配置する。カラーは青系。
2. 敵側パネル: 同等の配置で `Enemy Score` ラベル + 累積スコアを表示する。カラーは赤系でプレイヤーと差別化する。
3. 両スコアメーターはスペル枠と横並び（デスクトップ）または縦並び（小画面）で崩れないレスポンシブ対応を行う。
4. バトルモード中はヘッダーのグローバルスコア表示を非表示にする。
5. スコア加算ロジックに敵スコアの累積更新を追加し、演出（カウントアップ／SE）も共有する。
6. GAMEDESIGN に配置およびスコア加算仕様を追記する。

## テスト観点
- バトルモードでプレイヤー・敵ともにスコアが表示され、加点時に数値が増加する。
- ヘッダーのスコアはバトルモード中に表示されない。
- 画面サイズを縮小してもスコアメーターがスペル枠と重ならない。
- GAMEDESIGN 記載と実装が一致する。

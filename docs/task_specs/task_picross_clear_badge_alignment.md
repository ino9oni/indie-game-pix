# タスク仕様: ピクロス勝利数パネルの高さ調整

## 背景
- 現行レイアウトでは、ラウンド勝利数を表示するパネル（panel-score）がスペル枠より上下に大きく、バランスが崩れているとの報告がある。

## 目的
- 勝利数パネルの高さをスペル枠と揃え、上下ラインが揃った統一感のあるバトルUIにする。

## 要件
1. `.panel-score` の高さ・パディングを調整し、デスクトップレイアウトで `.panel-spell` と上端・下端が揃うようにする。
2. スペル枠と同じ角丸・境界線の見た目を維持しつつ、不要な余白を削減する。
3. 小画面レイアウト（縦並び）でも余白が過剰にならないようレスポンシブ調整を行う。
4. GAMEDESIGN のUI仕様に勝利数パネルの寸法調整を追記する。

## テスト観点
- デスクトップ幅でスペル枠と勝利数パネルの上下位置が揃う。
- モバイル幅でも余白が適切に収まり、はみ出しや重なりが発生しない。
- 調整によって他要素（進捗バー等）が崩れない。
- GAMEDESIGN 記載と実装が一致する。

# タスク仕様: マップモード背景画像の設定

## 背景
- 現在のランダム背景にはマップ専用画像が含まれていない。
- TASKLIST で「assets/img/background/map.png」をマップモード背景に指定する要求がある。

## 目的
- マップモードで常に `map.png` を背景として表示し、世界観と整合させる。

## 要件
1. マップモードでのみ `Background` コンポーネントに固定URLを渡し、`assets/img/background/map.png` を表示する。
2. 背景切替の仕組み（seed）を壊さないよう、他モードへの影響を避ける。
3. GAMEDESIGN に背景画像の固定化について追記する。

## テスト観点
- マップモードで背景が `map.png` に固定されている。
- 他のモードでは従来通りランダム背景が表示される。

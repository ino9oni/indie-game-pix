# タスク仕様: プロローグスキップボタンの追加

## 背景
- プロローグは一定時間ごとに自動スクロールする仕様だが、プレイヤーが任意のタイミングでスキップできない。
- TASKLIST の改善項目で、プロローグにスキップボタンを配置することが求められている。

## 目的
- プレイヤーがプロローグを即座に省略できるようにし、周回時の操作性を向上させる。

## 要件
1. プロローグ画面に「Skip」ボタン（ラベルは日本語でも可）を追加し、押下で即座に次のモードへ遷移させる。
2. 既存のテキストフェード演出・タイマーはスキップ時にすべて停止する。
3. スキップボタンはプロローグウィンドウ内の下部右寄せで表示する。
4. GAMEDESIGN でスキップボタンの存在が明記されていることを確認し、必要に応じて追記する。

## 非機能要件
- フォントスケール変更時にもレイアウトが崩れない。
- ボタンのフォーカス操作（キーボード操作）にも対応する。

## テスト観点
- ボタン押下でプロローグをスキップし、会話モード（あるいは次の画面）へ遷移する。
- 演出タイマーが残っていないこと（console log などで確認）。
- ボタンが画面下部に整列し、デザインと整合する。

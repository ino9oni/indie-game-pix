# タスク仕様: 名前入力画面の背景画像変更

## 背景
- 現状の名前入力モードはランダム背景またはマップ背景を参照しており、TASKLIST の背景指定と一致していない。
- 主人公の全身イメージを表示して世界観を強調する意図がある。

## 目的
- 名前入力モードで `assets/img/character/hero/hero_fullbody.png` を背景として表示し、画面演出を仕様と一致させる。

## 要件
1. 名前入力モード（`screen === "name"`、およびゲーム開始フローで `NameEntry` を表示している場合）において固定背景として上記画像を使用する。
2. 他モードの背景切り替えロジックに影響を与えない（ランダム or 既存の固定背景仕様を保持）。
3. GAMEDESIGN に背景変更の仕様を追記する。
4. 新しい背景でも名前入力 UI の視認性が確保されていることを確認する（カード内配色は現行のままで良い）。

## テスト観点
- 名前入力モード表示時に全身画像が背景として表示される。
- 他モードの背景表示が従来通りである。
- GAMEDESIGN と実装の内容が一致する。

# タスク仕様: タイトル画面背景の差し替え

## 背景
- 現状のオープニング（タイトル）画面では主人公の全身画像を背景に使用しているが、最新のアセット指示では専用のタイトル画像 `assets/img/title.png` を利用する必要がある。

## 目的
- タイトル画面の背景を指定のタイトル画像へ切り替え、ゲーム全体のビジュアルコンセプトと一致させる。

## 要件
1. `screen === "opening"`（タイトル画面）表示時の背景を `assets/img/title.png` に固定する。
2. 他モード（gamestart / name / route など）の背景切り替えロジックへ副作用を与えない。
3. `GAMEDESIGN` 文書のオープニングセクションを更新し、背景ファイルの指定を `assets/img/title.png` に修正する。

## テスト観点
- タイトル画面表示時に指定画像が背景として表示される。
- 名前入力やゲーム開始モードなど、他モードの背景が従来通り表示される。
- GAMEDESIGN の記述と実装が一致する。
